# Quick Calendar

External Qx plugin converted from the Raycast extension `calendar`.

## Source

```
https://github.com/raycast/extensions/tree/bf7fe09a27513c3d80ad02cfc7b152af1cc2c284/extensions/calendar
```

## Host capabilities used

- Raycast UI shim (`List` / preferences / navigation)
- `clipboard`, `open-url` (generic declaration; actual use is minimal)
- Preferences: week start, week numbers, compact/wide view

No binary HTTP or AppleScript is required. Works on Qx ≥ 0.4.28.
