# V2EX

Native Qx plugin for browsing V2EX topics, notifications, and token status.

## Features

- Latest / hot topic lists
- Node browsing (configurable node list preference)
- Notifications and token info (requires V2EX API token)

## Preferences

- **Access Token** — create at https://v2ex.com/settings/tokens
- **Nodes** — space-separated node names for the node tab

## Permissions

Uses exact `invoke:v2ex_*` host commands and `open-url`.
