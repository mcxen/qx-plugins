# QxHeihe 小黑盒

小黑盒社区浏览模块。使用 Qx Workbench 提供统一的搜索、列表、主从详情、Actions 和多图预览。

## 功能

- 推荐社区帖子列表与本地搜索
- 图片帖子筛选
- 点击或按 Enter 打开 Qx 内详情
- 异步加载完整正文和全部图片
- 可选登录后读取评论区，展示楼层、作者、时间、点赞与楼中楼摘要
- 点击图片使用 Qx 全尺寸预览
- 加载更多、刷新、缓存优先与离线回退
- 帖子已读/未读状态，并缓存完整详情供后续秒开
- 可选择 3 天或 7 天自动清理帖子、详情和已读状态
- 缓存占用接入 Qx「设置 → 关于 → 存储」，支持单独清理
- 在浏览器中打开原始小黑盒帖子

## 数据与限制

- 列表来自小黑盒 feeds API，详情来自公开的 `/bbs/web/link/detail`。
- 小黑盒评论接口要求登录。可在插件设置中粘贴完整 Cookie；Cookie 只作为评论请求头使用，不进入帖子缓存。未配置或失效时，正文仍可正常阅读并可跳转小黑盒评论区。
- feeds URL 带有小黑盒请求参数。如果默认地址失效，可在插件设置的 `Feed API URL` 中粘贴新的完整地址。
- 缓存使用 manifest `storage.cacheTargets` 白名单登记；清理缓存不会删除插件偏好设置。
