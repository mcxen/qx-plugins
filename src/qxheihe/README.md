# QxHeihe 小黑盒

小黑盒社区浏览模块。使用 Qx Workbench 提供统一的搜索、列表、主从详情、Actions 和多图预览。

## 功能

- 推荐社区帖子列表与本地搜索
- 图片帖子筛选
- 点击或按 Enter 打开 Qx 内详情
- 异步加载完整正文和全部图片
- 点击图片使用 Qx 全尺寸预览
- 加载更多、刷新、缓存优先与离线回退
- 在浏览器中打开原始小黑盒帖子

## 数据与限制

- 列表来自小黑盒 feeds API，详情来自公开的 `/bbs/web/link/detail`。
- 小黑盒评论接口要求登录，本插件只展示公开评论数量，不读取或保存登录凭据。
- feeds URL 带有小黑盒请求参数。如果默认地址失效，可在插件设置的 `Feed API URL` 中粘贴新的完整地址。
